# “躺平”问题的矛盾分析法解读

判断一个社会问题，首先要坚持实事求是。所谓实事求是，不是先有结论再找材料，也不是用一个方便的政治标签遮蔽复杂现实，而是从客观存在的事实出发，研究事实之间的内部联系。

围绕国家安全部将“躺平”话语归因于境外势力资助一事，最需要警惕的不是青年被几个短视频“洗脑”，而是社会治理在解释问题时偏离调查研究，把一个以就业、收入、劳动时间、分配预期为基础的现实矛盾，过早推入国家安全和敌我叙事之中。

本文的**基本判断**是：外部舆论力量可能利用中国社会焦虑，但它不是“躺平”情绪产生的主要根据。**“躺平”的根子在社会内部，在青年劳动者对高投入、低回报、低保障、低上升预期的现实感受之中。** 若不承认这一点，只在话语层面追查“境外组织”，就是离开了矛盾的主要方面。

## 一、先把事实摆出来

2026年4月28日前后，央视网等媒体转载国家安全部文章《煽动“躺平”的他们，正忙得脚不沾地》，另有央视网页面使用《某境外组织资助“躺平网红”企图渗透 国家安全部最新披露》这样的标题。公开文本称有“某境外组织”资助反华媒体、智库和“躺平网红”，系统性传播“躺平即正义”“反内卷=反剥削”等内容。

问题在于，公开报道没有披露组织名称、资金链条、具体账号身份、视频样本、传播数据和法律处置结果。严重指控需要扎实证据。没有这些细节，公众自然会追问：是谁资助？资助了谁？多少钱？造成了什么可验证后果？如果这些问题不能回答，“境外组织”就很容易在舆论中变成一个万能解释。

与之相对，青年压力却有大量可见事实支撑。国家统计局发布的2026年一季度数据中，全国城镇调查失业率平均值为5.3%，3月为5.4%，全国企业就业人员周平均工作时间为48.1小时；一季度全国居民人均可支配收入中位数为10433元。澎湃新闻转引国家统计局分年龄组失业率数据称，2026年3月不含在校生的16-24岁劳动力失业率为16.9%，25-29岁为7.7%，后者是这一口径自2023年12月发布以来最高。教育部此前预计2026届高校毕业生规模为1270万人。

这些数字说明，青年面对的不是抽象情绪，而是**真实的岗位竞争、收入约束和劳动强度**。即使宏观经济保持增长，个体层面的获得感也可能被**失业风险、超时劳动、住房压力、婚育成本和不稳定预期**抵消。离开这些事实去谈“奋斗意志”，很难说服人。

## 二、主要矛盾不在境外，而在内部

教员在矛盾分析中强调，复杂事物中有许多矛盾，但必须抓住主要矛盾。当前“躺平”争议中的矛盾至少有四组：

第一，国家发展叙事需要青年继续高强度投入，但青年看到的是努力回报下降、职业风险上升。

第二，官方把“躺平”解释为外部渗透，网民把“躺平”解释为就业难、低薪、加班、房价和阶层流动收窄的结果。

第三，宣传话语强调奋斗、奉献、民族未来，普通人关心双休、五险一金、工资、房租、婚育和职业安全。

第四，治理手段倾向于定性、封禁、管控，公众期待的是承认问题、解释问题、改善问题。

其中的主要矛盾，是发展动员对青年持续投入的要求，与青年在现实分配结构中获得感不足之间的矛盾。抓不住这个矛盾，其他问题都解决不了。即便封禁一个词，新的词也会出现；即便压下一轮评论，新的反讽也会出现。

“外因是变化的条件，内因是变化的根据。”从这个角度看，境外舆论如果存在，也只能借助内部矛盾发生作用。**没有就业焦虑、劳动权益焦虑和阶层流动焦虑，所谓“躺平叙事”就没有土壤。** 把外部条件说成根本根据，是把矛盾关系倒置了。

## 三、把人民内部矛盾当敌我矛盾，会扩大问题

多数“躺平”表达并不是组织化政治行动，而是一种低烈度的自我保护和情绪表达。有人说“我不卷了”，常常不是拒绝劳动，而是拒绝无偿加班、无效竞争和透支身体；有人说“我想躺平”，常常不是否定社会，而是对“努力也难以改变处境”的无奈概括。

这类矛盾首先应被看作人民内部矛盾。处理人民内部矛盾，不能靠扣帽子，更不能把群众的生活感受简单归入敌对势力操纵。正确的方法应当是调查、讨论、批评、改进，通过事实和政策重新建立信任。

如果把非对抗性矛盾对抗化处理，会出现三重后果。

**第一，真实问题被遮蔽。** 就业质量、劳动法执行、收入分配、公共服务和青年发展通道，本应成为政策讨论中心，却被“境外势力”这个标签挤到边缘。

**第二，公众经验被否定。** 普通人不需要境外组织告诉自己是否加班、工资是否够用、找工作是否困难。这些都是日常生活经验。若官方解释与日常经验冲突，公众不会认为自己错了，而会认为解释者脱离实际。

**第三，舆论反馈被异化。** 正常反馈渠道被压缩后，理性表达会减少，阴阳怪气、玩梗和反讽会增加。嘲讽不是问题消失，而是问题换了一种更隐蔽、更难治理的表达形式。

## 四、网民反讽是群众路线中的真实反馈

群众路线不是把群众当作被教育对象，而是承认群众拥有关于现实处境的第一手知识。网民那些看似戏谑的评论，其实包含着清楚的社会判断。

“给我安排个好工作，一举戳破境外势力阴谋”，说的是就业质量。

“如果我去烟草公司上班有编制，我会努力奋斗”，说的是体制内外、垄断行业与普通岗位之间的保障差异。

“落实双休、8小时工作制、五险一金，让境外势力无话可说”，说的是劳动法和社会保障。

这些话不是系统理论，却是群众经验的集中表达。它们共同指向一个判断：**青年并非天然反对奋斗，而是反对没有相应回报、没有基本保障、没有尊严边界的奋斗。**

真正的群众路线，应当把这些分散的、带情绪的、带玩笑色彩的表达集中起来，经过研究，转化为政策问题：哪些行业超时劳动最严重？哪些群体就业最困难？青年收入与住房、婚育、养老压力之间的缺口有多大？劳动监察为什么不足？职业上升通道为什么收窄？如果这些问题不问，只问谁在“煽动”，就是用命令主义替代群众路线。

## 五、问题不在“躺平”这个词，而在奋斗信用的流失

“奋斗”本来可以是积极词。问题在于，奋斗必须和可预期的改善相连。如果努力读书、努力工作、努力加班，换来的只是更高压力、更大债务、更不稳定的岗位和更晚的婚育，那么奋斗话语就会失去信用。

过去高速增长时期，许多人相信“多劳多得”“读书改变命运”“进城改变人生”。当经济增速放缓、房地产预期转弱、学历扩张、岗位竞争加剧之后，许多青年开始重新计算投入产出比。这种计算不是境外势力教出来的，而是社会实践教出来的。

实践是检验真理的标准。若宣传说“奋斗必有回报”，而大量青年在实践中感到回报不足，宣传就必须接受实践检验。此时需要修正的不是青年感受，而是奋斗机制本身：让劳动者有休息权，让就业有稳定性，让收入能覆盖基本生活，让普通人看见上升通道，让发展红利更公平地回到劳动者手里。

## 六、真正的反“躺平”不是反青年，而是改造产生“躺平”的条件

如果要从根本上削弱“躺平”叙事的吸引力，至少要做几件实事。

**第一，落实劳动法**，特别是工时、加班费、休息休假和劳动合同保护。最高法、人社部早已明确“996”严重违法，关键不在表态，而在执行。

**第二，提高青年就业质量。** 不能只看就业数量，还要看岗位稳定性、收入水平、社保覆盖、职业发展和专业匹配度。

**第三，改善分配结构。** 让普通劳动者能够分享发展收益，而不是只承担竞争成本。否则“共同富裕”会在舆论中变成空话。

**第四，降低住房、育儿、教育和养老压力。** 青年不婚、不育、低消费，往往不是观念突然堕落，而是成本和风险过高。

**第五，恢复表达空间。** 允许群众讲困难，允许青年说不满，允许媒体和学者研究真实问题。把问题讲出来，不等于给敌人递刀子；把问题压下去，才会让治理失去预警系统。

## 结语：回到实事求是，才是最大的安全

一个社会最可靠的安全，不是所有人都说好话，而是问题能被及时发现、真实表达、认真解决。把“躺平”简单归因于境外势力，看似维护了奋斗叙事，实际上削弱了叙事的可信度；看似保卫了青年思想，实际上否认了青年经验；看似压制了负面情绪，实际上让情绪转入更深的不信任。

正确路径并不复杂：没有调查，没有发言权；外因通过内因起作用；人民内部矛盾要用人民内部的方法解决；从群众中来，到群众中去；在实践中检验认识，再用新的认识改造实践。

因此，对“躺平”最有力的回应，不是寻找一个没有公布姓名的“某境外组织”，而是问清楚：为什么青年觉得努力不值？为什么正常双休会成为奢望？为什么编制和垄断岗位会成为反讽中的理想生活？为什么宏观增长不能稳定转化为微观安全感？

这些问题的答案**不在境外，主要在境内；不在短视频里，主要在劳动、分配和治理结构里**。真正实事求是地回答它们，才是重新凝聚青年、重建奋斗信用、维护社会长治久安的正道。
